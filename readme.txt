# Welcome to Express Typescript 
Express Typescript is version `Typescript` of `ExpressJs`. We will stop `Javascript` and go to `Typescript`.

## Explain 
This is important construct of source code.

### ./app 
We archive `controller` at here.
### ./resource/assets
We archive `statics` file at here.
### ./resource/views
We archive `view` at here. 
### ./routers
We archive `router` at here.
